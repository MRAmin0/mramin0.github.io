درباره من

من یک توسعه‌دهنده اپلیکیشن‌های موبایل هستم که با اشتیاق در اکوسیستم اندروید فعالیت می‌کنم. تمرکز اصلی من بر روی ساخت اپلیکیشن‌های Native با استفاده از جاوا و کاتلین است. از طراحی رابط‌های کاربری تمیز و کاربرپسند گرفته تا پیاده‌سازی معماری‌های نرم‌افزاری پایدار، همواره به دنبال یادگیری و به‌کارگیری بهترین روش‌ها برای خلق تجربه‌های دیجیتال باکیفیت هستم.

مهارت‌های فنی

زبان‌ها و UI

  • Kotlin
  • Java
  • XML Layouts
  • Jetpack Compose (در حال یادگیری)

معماری و الگوها

  • MVVM (Model-View-ViewModel)
  • Clean Architecture
  • Repository Pattern
  • Dependency Injection (Dagger, Hilt)

ابزارها و اکوسیستم

  • Android Studio
  • Gradle
  • Git & GitHub
  • RESTful APIs & Retrofit
  • Room Database

پروژه‌های من

اپلیکیشن یادداشت‌برداری

یک اپلیکیشن ساده و کارآمد برای ایجاد، ویرایش و مدیریت یادداشت‌ها با قابلیت ذخیره‌سازی محلی اطلاعات در دیتابیس Room.

تکنولوژی‌ها: Kotlin, MVVM, Room, XML

اپلیکیشن هواشناسی

اپلیکیشنی برای نمایش اطلاعات آب و هوا بر اساس موقعیت مکانی کاربر با استفاده از یک API عمومی و کتابخانه Retrofit برای مدیریت درخواست‌های شبکه.

تکنولوژی‌ها: Kotlin, Retrofit, Coroutines, XML
